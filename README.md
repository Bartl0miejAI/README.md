# 🌌 Other World's — Genesis Codex v1.1

> "In the Void, everything is possible. In Other World's – everything is real."

A living IP — an evolving Multiverse where AI generates unique worlds and blockchain guarantees their uniqueness. Each world is an NFT with its own DNA, history, physical laws, and Guardian.

**We don't sell pictures — we grant ownership of entire dimensions.**

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI-Powered Generation** | Worlds created by advanced LLM and diffusion models — each with unique DNA |
| ⛓️ **Blockchain + Web3** | NFTs (ERC-721/1155) with built-in logic limiting ownership to 1 world per wallet |
| 🪐 **81 Astro-Creationauts** | Cosmic engineers — a 9×9 matrix influencing each world's character |
| ⚡ **Impossible Laws** | Physical anomalies defining each world (e.g., reversed gravity, backward time) |
| 🔥 **Eternal Auction** | 3 modes: Stealth Drop, Gong Show, Bid War — with artwork reveal during bidding |
| 🗡️ **Convergence** | World duels — Winner Takes All. NFT burning, birth of the Nexus World |
| 📜 **Lore AI** | Worlds evolve narratively one year after birth — new stories, events, culture |
| 🏛️ **DAO** | 1 world = 1 vote — democracy without whale domination |
| 💎 **Deflationary Tokenomics** | Every Convergence permanently reduces world supply |
